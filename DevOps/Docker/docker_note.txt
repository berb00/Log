note:
    Docker的核心组件：
        Docker Client
        Docker daemon
        Docker Image
        Docker Registry
        Docker Container

修改docker仓库资源镜像路径
    1.编辑docker配置文件
        sudo vim /etc/default/docker
        底部加入:DOCKER_OPTS="--registry-mirror=https://registry.docker-cn.com"
    2.重启docker
        sudo service docker restart




commond:
查看docker版本信息
    docker version
    docker info

启动Docker-CE
    systemctl enable docker
    systemctl start docker
    service docker start

重启 Docker daemon
    systemctl daemon-reload
    systemctl restart docker.service

查看存在的镜像
    docker images
    docker image ls
查看容器
docker ps    // 运行的容器
docker container ls
docker ps -a    // 所有的容器
docker container ls --all
启动容器
    docker start container_name/container_id
    docker container start [containerID]
重启容器
    docker restart container_name/container_id
停止容器
    docker stop container_name/container_id
    bash container stop [containerID]
进入容器
    docker attach container_name/container_id
删除容器
    docker rm container_name/container_id
删除镜像    
    docker rmi image_name           // 需先删除镜像容器
    docker image rm image_name

从 Docker Hub拉取docker镜像:
docker pull image_name  // 拉取最新版
docker image pull hello-image_name
docker pull image_name:image_id     // 拉取指定版本

运行镜像
docker container run image_name

终止容器
docker container kill [containID]

查看容器输出
docker container logs [containerID]
