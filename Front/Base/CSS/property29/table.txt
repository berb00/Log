表格（Table） 属性

border-collapse 	规定是否合并表格边框 						2
border-spacing 	规定相邻单元格边框之间的距离 				2
caption-side 	规定表格标题的位置 						2
empty-cells 		规定是否显示表格中的空单元格上的边框和背景 	2
table-layout 	设置用于表格的布局算法 					2


border-collapse 		
JavaScript 语法: 	object.style.borderCollapse="collapse"
collapse 	如果可能，边框会合并为一个单一的边框。会忽略 border-spacing 和 empty-cells 属性		T:折叠，崩溃
separate 	默认值。边框会被分开。				  不忽略 border-spacing 和 empty-cells 属性		T:分隔，分离				
inherit 		规定应该从父元素继承 border-collapse 属性的值										T:继承
